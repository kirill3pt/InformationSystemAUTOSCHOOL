# Информационная система автошколы

Версия: 0.9.5  
Автор: Бережецкий К.Т.  
Год: 2025

---

## Описание

Данная программа предназначена для работы с модулями администратора и инструктора автошколы. Позволяет управлять данными об учениках, инструкторах, автомобилях и занятиях, формировать всё в один общий документ-отчёт

---

## Установка

1. Запустите файл установки `setup.msi`
2. Следуйте инструкциям мастера установки

---

## Использование

- Запуск программы из меню Пуск или с рабочего стола
- Выбор нужного модуля через меню (Администратор / Инструктор)
- Работа с таблицами: добавление, удаление, изменение записей
- Ведение журнала событий с возможностью выбора папки для логов

---

## Особенности

- Автоматическое выделение цветом статуса занятий
- Экспорт и импорт данных в формате Excel (.xlsx)
- Настройка путей для логирования
  
---

## Известные баги
- При выборе листа "РАСПИСАНИЕ" при отсустствии соотв. документа в папке, могут не появиться кнопки для отметки статуса занятия. Решение: снова выбрать лист "РАСПИСАНИЕ" сверху в выпадающем меню.
